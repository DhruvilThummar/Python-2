# Python-2: Data Analysis, Visualization, and Feature Engineering

This repository is a practical learning workspace for core data science topics using Python notebooks.
It is organized into three chapters:

1. Data analysis with Pandas and exploratory data analysis (EDA)
2. Data visualization
3. Feature creation and feature transformation

## What You Will Learn

- Loading and exploring real-world CSV datasets with Pandas
- Cleaning, transforming, and summarizing tabular data
- Performing basic EDA workflows
- Building meaningful visualizations to communicate insights
- Creating and transforming features for downstream modeling tasks

## Prerequisites

Install Python 3.9+ and the following libraries:

- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`
- `jupyter`

Install them with:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

## How to Run

1. Clone or download this repository.
2. Open the project folder in VS Code or Jupyter Lab.
3. Start Jupyter:

```bash
jupyter notebook
```

4. Open notebooks chapter-wise and run cells from top to bottom.

## Suggested Learning Path

1. Start with `Ch-1 Data Analysis with Pandas & EDA/Pandas(L1&L2).ipynb`
2. Continue through `Pandas(L3).ipynb` to `Pandas(L5).ipynb`
3. Practice with `PBSolve.ipynb`
4. Move to `Ch-2 Data Visualization/Data Visualization.ipynb`
5. Finish with `Ch-3 Feature Creation & Feature Transformation/Feature Creation & Feature Transformation.ipynb`

## Notes
